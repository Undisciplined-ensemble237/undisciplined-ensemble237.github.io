If you encounter trouble, use these solutions to resolve common issues.

*   No sound: Check that your Windows volume settings are not muted. Ensure your output device is selected correctly in the Sound Control Panel.
*   Registration prompts: If the software asks for a license, ensure you bought the full version from the authorized dealer. Enter the key in the settings menu under License.
*   Low file quality: Adjust the recording settings in the Options menu. Select higher bitrates if your storage space permits.
*   Installation block: If Windows Defender stops the program, click More Info, then click Run Anyway. This happens because the system does not recognize every new installer immediately.
*   File location: By default, the software stores files in the Music folder inside your user directory. You can change this path in Settings.
